About Mailcow Cleaner

Deletes messages from mailcow mailboxes matching sender, subject, age, and folder filters.

Uses doveadm expunge via docker exec for fast bulk deletion and the mailcow REST API for domain/mailbox listing.

Requirements

  • Docker socket (/var/run/docker.sock) mounted into the container
  • Mailcow API key with read-write access
  • Dovecot container accessible via docker exec

How it works

  1. List domains/mailboxes via mailcow API
  2. Preview matching messages using doveadm search
  3. Delete them using doveadm expunge
  4. Log results to history

Confirm Deletion

This will permanently delete message(s) from mailbox(es). This action cannot be undone.
